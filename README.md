# Módulo 3 – Reconocimiento Pasivo y Activo

Este repositorio contiene la actividad del Módulo 3 relacionada con técnicas básicas de reconocimiento en ciberseguridad.

## Contenido de la actividad

La actividad se dividió en cuatro partes principales:

### 1. Reconocimiento Pasivo con Google Dorks
Se realizaron diferentes búsquedas utilizando operadores avanzados de Google como:

- site:
- filetype:
- inurl:
- intitle:
- "frase exacta"
- OR
- exclusión (-)

Estas búsquedas permitieron identificar cómo información pública puede ser encontrada mediante técnicas de reconocimiento pasivo.

### 2. Investigación en Google Hacking Database (GHDB)
Se consultó la base de datos de Exploit Database para analizar diferentes Google Dorks y comprender qué tipo de información expuesta pueden detectar.

### 3. Análisis visual con Maltego
Se creó un grafo utilizando el dominio example.com y se ejecutaron transformaciones para identificar:

- Domain → DNS
- DNS → IP
- IP → Location

Esto permitió visualizar cómo se relacionan los elementos de infraestructura de un dominio.

### 4. Reflexión
Se realizó una reflexión sobre la diferencia entre reconocimiento pasivo y activo, los riesgos legales del reconocimiento sin autorización y la importancia de proteger la información pública.
